== Description ==

The Top Level Categories Fix plugin modifies the url which includes subcategories and paginations when the Top Level Categories plugin is in use.

== Installation ==

1. Copy the `top-level-cats-fix` directory into your `wp-content/plugins` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. That's it! :)
